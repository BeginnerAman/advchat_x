# AdvChat - Real-time Chat Application

A modern, interactive chat application built with Firebase Realtime Database. No signup needed—just enter your name and start chatting!

## Features

### Core Chat
- **Real-time messaging** - Messages appear instantly for all users
- **Online status** - See who's currently online
- **Typing indicator** - Know when someone is typing
- **Reply to messages** - Quote and reply to specific messages
- **Emoji reactions** - Quick react with emojis (❤️, 😂, 😮, 🔥, 👍, 😭, 🥰, 😎)
- **Emoji picker** - Full emoji grid for message input

### Admin Controls
Admins unlock special powers:
- **Edit messages** - Correct messages after sending
- **Delete messages** - Remove any message
- **Mute users** - Silence users for a set time
- **Global chat mute** - Mute entire chat
- **Pin messages** - Highlight important messages
- **Highlight messages** - Make messages stand out
- **Announcements** - Send system-wide announcements
- **Special effects** - Trigger hearts ❤️ or confetti 🎉

### UI/UX
- **Beautiful animations** - Smooth transitions and particle effects
- **Mobile friendly** - Responsive design for all devices
- **Auto-scroll** - Stays at latest messages
- **Keyboard shortcuts** - Enter to send, Shift+Enter for new line
- **Dark theme** - Easy on the eyes with modern colors
- **Sound effects** - Optional audio feedback

## Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Database**: Firebase Realtime Database
- **Animations**: GSAP (GreenSock Animation Platform)
- **Fonts**: Google Fonts (DM Sans, Syne)

## How to Use

1. **Open the app** - No installation needed, it's web-based
2. **Enter your name** - Any name you want (max 20 characters)
3. **Join chat** - Click "Join Chat" and start messaging
4. **Special roles** - Some names unlock admin features if you know the password

### Controls

- **Send message** - Press Enter or click send button
- **New line** - Shift+Enter
- **Reply** - Click the reply button (↩) on any message
- **React** - Click the react button (😊) to add emoji reactions
- **Online users** - Click 👥 to see who's online
- **Admin panel** - Click ⚙️ (admin only) for special controls
- **Emoji picker** - Click 😊 emoji button to add emojis

## Features in Action

### Messages
- See who sent each message
- Messages show timestamp
- Edited messages have ✏️ indicator
- Pinned messages have 📌 indicator
- Admin messages show 👑 badge

### Reactions
- Add reactions by clicking 😊
- Click reaction chip to toggle your reaction
- See how many people reacted and their names (hover)

### Announcements
- System messages show important events (joins, mutes, pins)
- Announcement toasts pop up as notifications

## Privacy

- No personal data is stored permanently
- Chat is public—anyone can see messages
- Messages stored in Firebase Realtime Database
- Your name and current status are visible to others
